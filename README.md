
Welcome to the snake game!


```
*****************************
*                           *
*                           *
*                           *
*         #                 *
*                           *
*         0                 *
*                           *
*                           *
*                           *
*                           *
*                           *
***************************** 
```

<br>Rules to play: <br>
    * The user can move in-between the * borders. <br>
    * The user aim is to eat the #.  <br>
    * Game ends hits a border. <br>
